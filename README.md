# DailyDrive Car Rental System

A Java-based management system designed to handle vehicle rentals, customer records, and transaction processing. This project was developed to demonstrate Object-Oriented Programming (OOP) design patterns and clean code practices.

## Key Features
- Dynamic Inventory Management: Support for multiple vehicle types, including Cars and Motorcycles.
- Rental Logic: Automated calculation of rental costs based on vehicle type and duration.
- Customer Management: Handling of customer profiles and rental histories.
- Input Validation: Error handling for user inputs and date selections.

## Tech Stack
- Language: Java
- Paradigm: Object-Oriented Programming (OOP)
- Tools: Git/GitHub for version control and Visual Studio Code for code editing and compiling  

## OOP Principles Applied
This project serves as a practical application of core OOP concepts:
- Inheritance: Used a base Vehicle class to share common attributes with specialized subclasses like Car and Motorcycle.
- Polymorphism: Implemented method overriding for cost calculations, allowing the system to determine prices dynamically based on the object type.
- Encapsulation: All data members are private, with access controlled through public getters and setters to ensure data integrity.
- Abstraction: Utilized abstract structures to define templates for all rentable items.

## Project Structure
- src/: Contains all .java source files.
- docs/: Includes the final project report and design documentation.
